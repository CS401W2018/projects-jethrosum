Internet

The Internet is an online network that enables computers and servers worldwide to connect and communicate.

Web

The Web is a system of interlinked data and information connected and accessed through the internet.

How are the Internet and Web different?

The Internet connects computers allowing for exchange of data, while the Web is a service that uses the Internet to allow for access to that data.

ASCII and Unicode

The American Standard Code for Information Interchange, or ASCII, and Unicode are systems for encoding characters in computers. ASCII is a subset of Unicode, meaning that Unicode may represent a much broader range of characters than other languages like ASCII.

What is UTF-8?

UTF-8 stands for Unicode Transformation Format - 8-bit, and it is a way to encode every Unicode characters in a computer. UFT-8 uses the mininum number of bytes necessary to encode a given character making it efficient and very common for storing and transmitting text in all languages.

URL

A Uniform Resource Locator, or a URL, is the complete web address by which resource on the internet is defined and located. A typical URL could be written as "http://www.google.com/index.html".

Domain Name

A domain name is the address used to locate a website on the internet. Some common domain names may include "google.com" and "wikipedia.org".

HTTP

Hypertext Transfer Protocol, or HTTP, allows the internet to function by loading web pages using hypertext links, an element that connects one document to another on the internet.

HTTPS

HTTPS stands for Hypertext transfer protocol secure, which is an encrypted version of HTTP to increase security of data transfers.

IP Address

IP Address means Internet Protocol address, and it is a unique numerical address for every network connected to the internet. An example of an IP address is 192.168.252.199.

DNS

A Domain Name System or DNS translates domain names to IP addresses that allow browsers to locate and load web pages and other online resources.

W3C Recommendations

The W3C, or World Wide Web Consortium, is an international organization that creates and enforces guidelines for the web. W3C recommendations include technical specifications and best practices created by the W3C that ensures the internet remains user-friendly, safe, and compatible across different platforms.

XML, XHTML, HTML, HTML5

XML, XHTML, HTML and HTML5 are markup languages with varying uses. XML is a language used to store and transfer data. XHTML is a stricter version of HTML that structures webpages. HTML also structures web pages but doesn't require strict formatting. HTML5 is the latest version of HTML used to create modern websites with multimedia.
