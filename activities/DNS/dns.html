  • What is DNS?
    Domain Name System - the phonebook of the internet. This translates IP addresses into human-readable names.
  • How do you get a domain name?
    We can get a domain name by working with a registrar such as GoDaddy to register a domain name.
  • What is SSL?
    This stands for Secure Sockets Layer, which is protocol that encrypts data and creates a secure connection between one's device and the website or server.
  • How do you get a security certificate?
    Security certificates prevents hackers from obtaining personal data when our personal device is communicating with a website or server. To obtain a certificate, one must request for one on one's server.
  • How does SSL work?
    We use security keys (eg. DEZ) to encrypt and decrypt our data to prevent others from obtaining our data in its original readable form. If the public key encrypts data, then the private keys decrypts data, or vice versa. Installed in our operating systems, we get a list of certificate authorities we trust. We should rely on this list to ensure that our data is safe.
