Engagement Questions (Nov 1)

  • Why don't we use the border property?
    We don't use the border tag within html because it's now invalid. We add borders to our table using CSS instead.
  • What is the easiest way to color every other row a different color?
    We can use this 'tr:nth-child()' as an easy way to color every other row a different color.
  • When should we use a table?
    We use tables when we need to organize data or information included in our webpage in a clear order.
