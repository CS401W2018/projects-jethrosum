    • When should you use a background image, and when should you use the IMG tag?
      • A background image is used in CSS when the image being included isn't especially important to the content of the webpage. Using CSS to include a background image also provides greater flexibility with positioning. IMG tags are used within HTML files when the image being included is essential to the content.
