Engagement: CSS Selectors

1. List out the different ways to select an item.

(1) Tag Name, (2) Descendent (eg. main p), (3) ID and (4) Class

2. What is the difference between an ID and a Class?

An ID allows us to select all the tags and apply changes to all those tags. For example, (within a tag) id = "first". This name has to be spelled the same way between the html and the css files. You also cannot have any space between multi-word names; only hyphens. To use modify it in the css file, use # then the id name. For example, #first. An ID can also only be used ONCE. With an ID, each tag can only have ONE.

A class works almost the same way as an ID but instead of using a hashtag/pound sign, use a period (.). A class can also be used as many times as you want. A tag can also have many classes.

4. What is the difference between an DIV tag and HEADER, MAIN, FOOTER?

Header, Main, and Footers tags essentially acts the same way as DIV tags would. It simple makes it easier to locate each section of a web page code.

5. What are different ways to select a color?

We can select a color by calling it's specific name, selecting the color using a box color selector, and by inputting its color code.
