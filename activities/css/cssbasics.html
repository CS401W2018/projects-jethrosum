Engagement Questions

1. Explain the difference between inline, embedded, and external CSS.

An inline CSS defines a single element with the insert style sheets in an HTML document. An embedded CSS is style sheet within the head tag that defines specific elements within the HTML document. An external CSS is a separate CSS document that is linked to an HTML document to define its elements.

2. Explain why you should only use external CSS.

External CSS allows coders to avoid a greater workload and helps keep their codes neat and simple.

3. What is a selector used for?

Selectors are used to find or choose the specific HTML element coders want to modify.
