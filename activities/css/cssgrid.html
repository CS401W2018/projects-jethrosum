Engagement - CSS Grid

  • How much time should you spend on an engagement assignment after class?
    Around 20 to 30 minutes.
  • What should you focus on during class and after class?
    We should focus on following along the code being created in class and filling out the engagement assignment along the way.
  • What is the main difference between CSS Grid and CSS Flex?
    CSS Grid is two dimentional while CSS Flex is supposed to be one dimentional. In CSS Grid, the parent determins the size of the children, while CSS Flex, the children determines it's own size and the parent works around it.
