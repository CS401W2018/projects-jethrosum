Engagement - October 23

  • Why is the viewport meta tag important?
    It ensures web pages display correctly on different screen sizes, especially on mobile devices.
  • How can you adjust the CSS for printing?
    Use @media print in CSS to define styles specifically for printed versions of your webpage.
  • When would you want to adjust the CSS for printing?
    You'd adjust it when you want your webpage to look clean and optimized when printed, such as hiding navigation menus or simplifying layouts.

Engagement - October 25

  • Why do we need a viewport meta tag?
    The viewport meta tag allows our media queries to work and ensures that web pages diplay corectly on various screen sizes.
  • When you select the same tag twice, and in one of them you say "flex-direction: column;" and the other says "flex-direction: row;", which one takes priority?
    The second one will take priority as it is the latest command.
  • How do you change the CSS for a web page displayed on a phone?
    Include the "@media only screen" on your Css to change the CSS for a webpage displayed on a phone.
