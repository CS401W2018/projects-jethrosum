Engagement Questions - Nov 6

  • The form tag has an "action" attribute. What does it do?
    The action attribute defines what will be done after the form is submitted.
  • The form tag has a "method" attribute. What does it do?
    This attribute specifies how the form data will be sent, either using POST or GET.
  • What is the difference between a "POST" and a "GET"?
    "GET" displays the inputted information with the domain name when it's submitted, while "POST" hides its visibility but doesn't make it more secure.

Engagement Questions - Nov 8

  • What are different ways to style your form?
    We can use CSS to control input sizes, colors, and add things like rounded borders for a clean look. I can also use focus effects on buttons to make the form feel more interactive.
  • Why does the professor like to use flex?
    Flex allows webpages to adjust easily to small screen sizes.
