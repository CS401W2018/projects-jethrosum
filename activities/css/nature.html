Engagement Questions

  • What is the difference between a block and an inline tag? Give examples of each one.
    Block tags' width and height is determined by the amount of content within the html. Block tags also always starts from a new line, while inline tags don't and only cover the space as bounded by the tags in the HTML element.
  • What is normal flow in HTML?
    It's how block and inline tags are used to create the default way of organizing HTML.
  • What does float do? Why do you need to clear the float?
    The float tag allows us to position elements in certain places in our webpage. Clearing the float will position all other elements in their correct positions.
  • What do absolute, fixed, and relative do?
    Relative slides things around but reserves the space. Absolute makes other elements in the webpage ignore the element which this tag is applied to. Fixed fixes the elements in the edge of the webpage. Absolute and fixed is typically used in headers and footers.
  • How can you make nice navigation buttons for mobile devices?
    By using "hover" in css.
