Block Tags

Text Text Text Text Text Text
  • What is the difference between block tags and inline tags?
    • Block tags' width and height is determined by the amount of content within the html. Block tags also always starts from a new line, while inline tags don't and only cover the space as bounded by the tags in the HTML element.
  • What is the margin, border, and padding?
    • The margin the very outer layer of the page. The border is the layer between the padding and the margin. The padding is the layer nearest the content and lies between the content and border.
  • What are the units of measurement you can use in CSS?
    • Pixels are the most commonly used unit of measurement in CSS.
  • When would you use a figure tag?
    • The figure tag is used to group HTML media elements including images, diagrams and illustrations.
