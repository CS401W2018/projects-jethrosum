  • What is JavaScript? How did it get its name?
    The most commonly used programming language. Java was created by Sun Microsystems. The creators of JavaScript used Java and wanted to implement and use it within Sun Microsystems' internet browser, NetScape, so they created JavaScript. It was a marketing ploy to pull Java into the web.
  • What are the two ways you can run JavaScript?
    1. In the browser or in the "DOM"
    2. Through Node.js (Run JavaScript as you would Python)
  • What was the most popular front-end js library?
    JQuery.js was the most popular front-end js library.
  • What is the most popular front-end js library today?
    React.js is the most popular front-end js library today. This library was created by Meta, the parent company of Facebook and Instagram.
